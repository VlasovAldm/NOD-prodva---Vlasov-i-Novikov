# На данный момент реализован парсер данных с ВК - для анализа целевой аудитории, парсер с Авито - для анализа "поставщиков", товаров для продажи.






## Название
Перевоз.ру

## Состав группы
Новиков Тимофей (github: --), Власов Александр (github: VlasovAldm)

## Идея компании
Компания по поиску разности в ценах машин в разных регионах 

Позволяет перевозчикам машин быстрее искать разницу в ценах и привозить их в город с более высокой ценой или купить для личного использования по более низкой цене.

Ищет объявления на продажи в разных городах.

Основной потребитель - это люди, которые хотели бы купить машину дешевле в другом регионе.

Тимофей Новиков - анализ целевой аудитории, маркетинговая стратегия
