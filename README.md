## Идея компании
сервис по поиску всей информации по конкретному спортивному событию. 

Даёт более глубокое погружение в контекст мероприятия, чем самостоятельный ресерч.


Кто играет, цена билета, соцсети клуба, игроков, где можно записать ребенка на секцию по этому виду спорта, коэффициенты для ставок.

Основной потребитель - люди, которые увлекаются спортом, хотят сходить на мероприятие.
